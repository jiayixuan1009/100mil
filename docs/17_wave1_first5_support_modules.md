# Wave 1 第一批 5 页 support modules 说明

## 文档用途

本文件说明 `phase2_wave1_first5_support_modules.csv` 的用途。

这个文件用于把第一批 5 页的支持模块拆给不同角色：

- 内容：FAQ 主题
- 产品：CTA 模块
- SEO / 运营：内链目标

## 当前文件

- `phase2_wave1_first5_support_modules.csv`

字段包括：

- url
- primary_query
- faq_topics
- cta_modules
- internal_link_targets
- owner

## 使用方式

1. 先用 [16_wave1_first5_copy_direction.md](16_wave1_first5_copy_direction.md) 确定标题和首屏方向。
2. 再用 `phase2_wave1_first5_support_modules.csv` 完成 FAQ、CTA 和内链模块。
3. 对技术仍有门槛的页面，support modules 可以先准备，但不应先上线。

## 当前执行结论

- 页面 3、4、5 已经具备进入第一波内容改造的基础条件。
- 页面 1、2 应先解决技术问题，再决定上线顺序。
