# Converter 模板块级清单

## 用途

本清单用于将 Converter 模板执行规格转化为块级任务，方便内容、产品、前端协作。

## 必备块

### 1. SEO 基础块

- Title
- Meta Description
- H1
- Canonical
- 可索引状态

### 2. 汇率结果块

- 币种对结果
- 更新时间
- 汇率说明
- 常用金额切换

### 3. 用途说明块

- 汇率结果如何理解
- 是否包含手续费
- 转账或换汇场景解释
- 大额/小额差异说明

### 4. 关系页面块

- 逆向币对
- 相邻币对
- compare 页面
- sendmoney 页面

### 5. FAQ 块

- 汇率多久更新
- 转账实际到账与显示汇率差异
- 大额金额注意事项
- 费用或 spread 影响

### 6. CTA 块

- 主 CTA：Check Live Rate / Start Transfer
- 次 CTA：Download App / Compare Best Rate

## 协作分工

- SEO：Title、Meta、关键词和 SERP 匹配
- 内容：汇率解释、FAQ、首屏说明
- 产品：金额 chips、结果展示、CTA 路径
- 前端：结果区、更新时间、相关页组件

## 发布前检查

- 首屏是否先展示结果而不是文案
- amount 示例是否足够直观
- 是否补了 compare / sendmoney 关系页
- CTA 是否可追踪
- Title 和 Meta 是否过长
