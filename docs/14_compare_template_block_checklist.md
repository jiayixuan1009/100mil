# Compare 模板块级清单

## 用途

本清单用于将 Compare 模板执行规格转化为前端、产品、内容、SEO 可协作的块级任务。

## 必备块

### 1. SEO 基础块

- Title
- Meta Description
- H1
- Canonical
- 可索引状态

### 2. 首屏结果块

- 币种对标题
- amount 展示或输入
- 当前最佳结果摘要
- 更新时间
- 简短价值说明

### 3. 对比说明块

- 汇率结果说明
- spread / 费用提示
- 到账时间或处理时间说明
- 使用场景提示

### 4. 导航与内链块

- 对应 converter 链接
- 相邻 amount compare 链接
- 相邻币种 compare 链接
- sendmoney corridor 链接

### 5. FAQ 块

- 最佳汇率来源
- 金额变化是否影响结果
- 手续费和隐藏价差
- 更新时间和可信度

### 6. CTA 块

- 主 CTA：Check Live Rate / Start Transfer
- 次 CTA：Download App / Compare Another Amount

## 协作分工

- SEO：Title、Meta、H1、关键词映射
- 内容：结果说明、FAQ、首屏摘要
- 产品：amount 逻辑、结果解释、CTA 路径
- 前端：块位、更新时间、内链模块实现

## 发布前检查

- 页面是否可索引
- canonical 是否正确
- 首屏是否直接回答对比问题
- 是否有 amount 和结果语境
- CTA 是否可追踪
