## 当前原则

- 先修数据口径，再做增长动作。
- 先优化已有曝光页面，再决定是否新增页面。
- 先找真实可运营流量，再追求总流量数字。
- 每个阶段都输出可执行表格、判断标准和下一阶段输入。
- 每个专项和阶段都要形成独立文档，便于复盘、协作和汇报。
